# WMS Pro

Struktur proyek:

- `index.html` — halaman utama.
- `css/style.css` — semua gaya tampilan.
- `js/app.js` — logika aplikasi, routing, rendering halaman, dan interaksi.
- `js/db.js` — lapisan data lokal (LocalStorage) dan fungsi CRUD.
- `images/` — asset gambar untuk slider dan ilustrasi.
- `database/wms_database.sql` — file SQL cadangan untuk struktur data.

Penggunaan:
1. Buka `index.html` di browser atau melalui server lokal.
2. Masuk dengan username `faisal` dan password `admin123`.
3. Dashboard akan memuat data dan notifikasi transaksi.
